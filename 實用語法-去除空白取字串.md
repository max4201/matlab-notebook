# MATLAB實用語法
## MATLAB去除空白取字串
```matlab
%--------------------------------------------------------------------------
% 把所有不限數量空格分開的字串取出
Target_String=['   56   74   29    0 LOGE'];
% 將目標字串依序匹配所有連續非空格的字符並輸出。等效於把所有空格分開的字串取出。
% 「\S」指空白符號，後面接著「+」就是1~Inf個。
Output_cell=regexp(Target_String,'\S+','match');
%--------------------------------------------------------------------------
```
