# MATLAB實用語法
## MATLAB圖置中移動
**使用平台:**
Windows 10 64bit

**MATLAB版本:**
MATLAB R2013a 64bit

**說明:**
使用movegui函數，同樣也可以移動到其他位置，細節請參閱Matlab help。


**Matlab Script:**
```matlab
%-----------------------------------------
%清除變數，清除螢幕，清除所有figure。
clear;clc;close all
%-----------------------------------------
% 建立figure並設定為不可視
figure('Visible','off');
% 移動到螢幕中間
movegui(gcf,'center')
% 改為可視
set(gcf,'Visible','on');
%-----------------------------------------
```
